Indian Coffee Beans — India's Specialty Coffee Directory

India's first comprehensive, independent directory for specialty coffee roasters and their offerings. A research-friendly resource that helps home brewers discover, compare, and learn about Indian specialty coffee — plus brewing tools and a public REST API.

Who It's For

  • Home brewers and specialty coffee enthusiasts in India
  • People discovering and comparing Indian roasters and coffee beans
  • Developers building on the public coffee/roaster API

Important Pages

  • Coffee Directory — browse and filter Indian specialty coffees
  • Roaster Directory — discover Indian specialty roasters
  • Learn — Field Guide & Educational Hub
  • Coffee Tools — calculator, coffee compass, expert recipes
  • About
  • How ICB Works (transparency & neutrality)
  • Developer API Docs

Public API

  • `/api/coffees`
  • `/api/coffees/filter-meta`
  • `/api/roasters`
  • `/api/roasters/filter-meta`
  • `/api/search-index`
  • Versioned, API-key-authenticated endpoints under `/api/v1/` (coffees, roasters, reviews, users, usage) — see Developer API Docs

Key Facts

  • Category: directory / reference (specialty coffee)
  • Region: India
  • Audience: home brewers, coffee enthusiasts, developers
  • Independence: independent and neutral — not a marketplace or store
  • Contact: https://www.indiancoffeebeans.com/contact
